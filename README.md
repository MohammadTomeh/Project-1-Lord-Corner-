# 🛍️ Lord Corner

A modern e-commerce website for men's fashion, featuring sneakers, hoodies, and streetwear collections.

---

## 🚀 Live Demo

🔗 https://mohammadtomeh.github.io/Project-1-Lord-Corner-/

---

## 📌 About the Project

Lord Corner is a front-end project built to showcase a clean and modern e-commerce layout.

The goal was to build a structured and visually appealing shopping experience using only HTML and CSS, without relying on modern layout systems like Flexbox or Grid.

---

## 🛠️ Built With

* HTML  
* CSS (without Flexbox or Grid)

---

## ✨ Features

* Clean and modern UI design  
* Desktop-first layout design  
* Multiple sections (Hero, Features, Products, Offers, Pricing, Blog, Team)  
* Smooth hover effects and animations  
* Organized layout following a real e-commerce structure  

---

## 📈 What I Learned

* Structuring a real-world website layout  
* Writing clean and organized CSS  
* Designing consistent UI components  
* Using Git & GitHub in a real workflow  

---

## ⚠️ Notes

* This project is a front-end design only (no backend functionality)  
* Some buttons are for UI demonstration purposes  
* Currently optimized for desktop view — responsiveness will be added in future updates
